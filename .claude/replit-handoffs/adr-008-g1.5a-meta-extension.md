## Context

ADR-008 extended `AnalystVerdictMetaSchema` with three optional fields (`fallbackReason`, `vendorsUsed`, `cacheState`) plus three new Zod refinements coupling each field to its tier. This packet implements ADR-008 in the engine and emits `fallbackReason: "tier1_unavailable"` from the Funding Specialist's Tier-0 fallback path so Replit's behavioral verification of G1 can pass.

The work is constrained to the **Tier-0 fallback half** of G1's verification spec. Real Tier-1 emission (`vendorsUsed`, `cacheState`, `cognitiveRunId`) will fire only after Replit's G1.5c packet wires `FundingSpecialistDeps` at the registration site. The schema will already accept those fields; they'll start arriving as soon as deps are wired.

Owner: CC (research/intelligence lane per `claude-replit-split.md` 2026-04-26).
