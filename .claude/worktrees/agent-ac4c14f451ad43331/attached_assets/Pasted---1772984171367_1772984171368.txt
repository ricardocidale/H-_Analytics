                                                                                                                                                                                                                                                                              
  What was built                                            

  New Admin tab: "Research" under the Business nav group in Admin → accessible at /admin → Research sidebar item.

  Per-event control for all 3 research types (Property / Company / Global):
  - Enable/disable toggle — disabled types return 403 when triggered
  - Focus Areas — tag-style input, injected as bulleted guidance into the prompt
  - Geographic Regions — tag-style input, scopes the research geography
  - Investment Horizon — dropdown (5/7/10/15-year)
  - Custom Instructions — free-text appended to the system prompt
  - Custom Questions — specific questions the LLM must address
  - Deterministic Tools — checkboxes for all 9 tools (property type only); unchecked all = all tools enabled
