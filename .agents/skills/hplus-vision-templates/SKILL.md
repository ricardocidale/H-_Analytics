# H+ Analytics — Vision Text Templates

All templates are deterministic — no LLM required. They produce professional, investor-grade slide copy from structured property data alone. The LLM in `property-vision.ts` uses these as fallbacks and as a formatting guide for its own output.

---

Every property deck needs these fields. All are generated by `property-vision.ts` (LLM + fallback):

```python
def get_vision_templates(property_data: dict) -> dict:
    """
    Returns deterministic vision text for a property.
    Falls back to 'hotel' tier if type not recognized.
    """
    model = (property_data.get("businessModel") or
             property_data.get("hospitalityType") or "hotel").lower()
    tier = "retreat" if "retreat" in model else \
           "vrbo"    if model in ("vrbo", "vacation_rental", "airbnb") else \
           "hotel"
    tmpl = TEMPLATES[tier]
    return {k: v(property_data) for k, v in tmpl.items()}
```

## LLM Prompt Guidelines (for property-vision.ts)

When the LLM generates vision text, it MUST:
1. Use the template outputs as the **format reference** — match field lengths, capitalization style, and tone exactly
2. **Cite actual property data**: ADR, room count, purchase price, location — no generic placeholders
3. **Never use**: "exciting opportunity", "strong fundamentals", "world-class", "unique"
4. **Always use**: specific numbers, measurable outcomes, named demand drivers (e.g., "Belleayre ski area", "2.5hr NYC drive-market")
5. **Cinematic caption format**: `"[FEATURE] · [FEATURE]"` — ALL CAPS, concise, visual (max 2 features)
6. **Badge text**: 3–5 words, ALL CAPS, describes the experience brand promise

## Renovation Rationale Text by Scope

```python
RENOVATION_RATIONALES = {
    "light":   "Cosmetic refresh targeting quick-to-market timeline: design, photography, and OTA optimization.",
    "medium":  "Strategic renovation to unlock key revenue channels: en-suite upgrades, common area refresh, and F&B infrastructure.",
    "full":    "Full conversion buildout: en-suite guestrooms, commercial kitchen, spa amenities, and direct booking infrastructure.",
    "historic": "Historic preservation-grade renovation balancing character retention with modern hospitality standards.",
}

def get_renovation_rationale(scope: str, is_historic: bool) -> str:
    if is_historic:
        return RENOVATION_RATIONALES["historic"]
    return RENOVATION_RATIONALES.get(scope, RENOVATION_RATIONALES["medium"])
```
