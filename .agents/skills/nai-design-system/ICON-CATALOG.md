# NAI Icon Catalog

All icons are imported from `@/components/icons/` — never directly from `@phosphor-icons/react`.

**Icon library:** `@phosphor-icons/react` v2.1.7 (Phosphor Icons)

**Barrel file:** `artifacts/hospitality-business-portal/src/components/icons/index.ts`

Many icons have a `*Icon` alias (e.g. `IconPlus` → `IconPlusIcon`). These aliases are identical references; prefer the base name.

## Adding a New Icon

1. Find the icon name on [phosphoricons.com](https://phosphoricons.com/).
2. Add the Phosphor export to the appropriate category source file (e.g. `action-icons.tsx`) as `PhosphorName as IconYourName`.
3. Re-export `IconYourName` from `brand-icons.tsx` (via its category file's `export *`).
4. Add a corresponding entry to this catalog in the correct category table: `| \`IconYourName\` | \`PhosphorName\` | Brief visual description |`.
5. If you are creating a semantically identical alias, add it to the Alias Map table above.
