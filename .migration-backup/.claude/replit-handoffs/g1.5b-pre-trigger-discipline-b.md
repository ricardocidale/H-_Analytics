# G1.5b-pre-b: Client — remove auto-trigger hooks + repurpose dialog

## Doctrine Freeze Gate Check

- **Governing rule:** [`.claude/rules/analyst-trigger-discipline.md`](../rules/analyst-trigger-discipline.md), Binding 2026-04-26.
- **Gate decision:** ✅ Cleared (bugfix-against-shipped-code exception).

## Context

After G1.5b-pre-a removes server-side dispatch from save-tab, the client must:
1. Stop expecting `verdict` in the save-tab response.
2. Stop firing The Analyst from URL-param deep links and auto-refresh hooks.
3. Display a "Due for review" badge instead of auto-firing when intelligence goes stale.

Parent: [`g1.5b-pre-trigger-discipline.md`](./g1.5b-pre-trigger-discipline.md).

Owner: **Replit**. Depends on G1.5b-pre-a being shipped first.
