# Property Conversion Pipeline

## The Pipeline
1. Buy large home on large lot (estate, ranch, farm)
2. Main house → common areas (restaurant, bar, lobby, event space)
3. Add rooms: A-frame cabins ($75-130K/key), glamping ($35-70K/key), outbuildings
4. Barn/outbuilding → event venue ($300-750K)
5. Commercial kitchen ($250-450K)
6. Zoning, permitting, fire code, ADA, liquor licensing
7. Pre-opening: staffing, training, marketing, soft launch
8. Start operations

## Cost Per Key (US)
| Type | Cost/Key |
|------|----------|
| Main house conversion | $200K-400K |
| A-frame cabin (all-in) | $75K-130K |
| Glamping unit (all-in) | $35K-70K |

## Cost Per Key (Colombia)
| Type | Medellín | Cartagena (heritage) |
|------|----------|---------------------|
| Conversion | $80K-180K | $120K-250K |
| A-frame | $35K-70K | Same |
| Glamping | $15K-35K | Same |

## Timeline
- US: 18-36 months (24 typical)
- Colombia (Medellín): 12-24 months
- Colombia (Cartagena walled city): 24-36+ months (heritage restrictions)

## Constraints Research Must Flag Per Property
- **Septic capacity** — binding constraint on rural room count
- **Corridor width** — 36" residential vs 44" commercial (mansion deal-breaker)
- **Utah water rights** — commercial use requires different rights than residential
- **Utah liquor quotas** — 40+ room threshold for hotel license, bar licenses $50K-500K+
- **NY SEQRA** — can add 3-12 months
- **Cartagena heritage** — Ministry of Culture approval required
- **Under 12-15 rooms** — struggle to cover fixed operating costs
- **Elevator** — required 3+ stories ($75-200K)
- **Insurance** — US: $75K-150K/year; Colombia: $12K-30K/year

## Successful Examples
- Troutbeck (Amenia, NY): 250-acre estate → 37 rooms, $500-800K/key, ADR $400-800+
- Hasbrouck House (Stone Ridge, NY): 50-acre → 19 rooms multi-building, $260-420K/key
- Inness (Accord, NY): 225-acre dairy farm → 28 cabins, ADR $400-700+
- Camp Sarika at Amangiri (Utah): luxury tented pavilions at $3-8K/night
- Casa San Agustín (Cartagena): heritage houses → 31 rooms, ADR $400-800+

## Key Lesson
Main house is most valuable as common areas. Multi-building estate format (main house + additions) allows phased development and room product variety.
