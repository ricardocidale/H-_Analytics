You are tasked with implementing a professional three-font typography system for this H+ Analytics financial application based on Swiss Modernist design principles.

## Context

This application currently has typography inconsistencies. I have comprehensive documentation that defines the complete typography system. Your job is to implement it precisely.

## Documentation Files

Read these files IN ORDER before making any changes:

## The Three-Font System

**IBM Plex Sans (font-display)** - ALL headings, titles, card labels, section headers
**Inter (font-body)** - ALL body text, labels, buttons, form inputs, UI text
**JetBrains Mono (font-mono)** - ALL numbers, currency, percentages, metrics, dates

## Critical Implementation Requirements

### 1. Font Loading (src/styles/fonts.css)

```css
@import url('https://fonts.googleapis.com/css2?family=IBM+Plex+Sans:wght@400;500;600;700&display=swap');
@import url('https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600&display=swap');
@import url('https://fonts.googleapis.com/css2?family=JetBrains+Mono:wght@400;500;600&display=swap');

@layer utilities {
  .font-display {
    font-family: 'IBM Plex Sans', sans-serif;
  }

  .font-body {
    font-family: 'Inter', sans-serif;
  }

  .font-mono {
    font-family: 'JetBrains Mono', monospace;
  }
}
2. Theme Configuration (src/styles/theme.css)
Add font size variables to :root:

Body element MUST use Inter:

ALL headings MUST use IBM Plex Sans:

  font-family: 'IBM Plex Sans', sans-serif;

3. Root Container (src/app/App.tsx)
The root div MUST include font-body:

4. AnimatedCounter Component
This component MUST default to font-mono. Update src/app/components/AnimatedCounter.tsx:

const finalClassName = className
  ? className.includes('font-')
    ? className
    : `font-mono ${className}`
  : 'font-mono';

return (

Common Mistakes to Avoid
❌ DO NOT use @apply font-body in theme.css (Tailwind v4 incompatible) ✅ DO use font-family: 'Inter', sans-serif; directly

❌ DO NOT forget font-mono on table number columns ✅ DO add className="font-mono" to every with numbers

❌ DO NOT use font-display on body text ✅ DO use font-display ONLY on headings/titles

❌ DO NOT skip the root container font-body class ✅ DO add font-body to App.tsx root div
