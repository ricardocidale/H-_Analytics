  ---                                                                                                      
  New API endpoints (all require admin):                          

  POST   /api/lb-slides/factory/runs
  GET    /api/lb-slides/factory/runs                                                                       
  GET    /api/lb-slides/factory/runs/:id
  POST   /api/lb-slides/factory/runs/:id/brief                                                             
  POST   /api/lb-slides/factory/runs/:id/accept-brief                                                      
  POST   /api/lb-slides/factory/runs/:id/properties
                                                                                                           
  Run shape returned by all endpoints:                                                                     
  {
    id: number                                                                                             
    userId: number                                                
    status: "new" | "brief_ready" | "ingesting" | "ingested" | "drafting" | "draft_review" | "building" |
  "complete" | "error"                                                                                   
    briefR2Key: string | null                                                                              
    briefFilename: string | null
    briefAccepted: boolean                                                                                 
    canonicalSpec: unknown | null       // set by Lorenzo (Tab 2) 
    canonicalPngKeys: string[] | null   // set by Lorenzo (Tab 2)                                          
    slide1PropertyId: number | null                                                                        
    slide2PropertyId: number | null                                                                        
    slide3PropertyId: number | null                                                                        
    slide5PropertyId: number | null                               
    luccaDraft: Record<string, unknown> | null    // set by Lucca (Tab 4)                                  
    agentResults: Record<string, unknown> | null  // set by agents (Tab 5)
    deckR2Key: string | null                                                                               
    startedAt: string | null
    completedAt: string | null                                                                             
    createdAt: string                                             
    updatedAt: string                                                                                      
  }                                                               

  ---                                                                                                      
  Tab 1 — Brief
                                                                                                           
  Gate: show when status === "new"                                
                                                                                                           
  1. On tab load: GET /api/lb-slides/factory/runs — if no run exists or the latest is complete/error, show 
  a "Start new run" button → POST /api/lb-slides/factory/runs to create one.                               
  2. File upload: use the existing presigned URL flow — POST /api/uploads/request-url (body: { name, size, 
  contentType, entityType: "slide_factory_brief" }) → upload directly to R2 → call POST                    
  /api/lb-slides/factory/runs/:id/brief with { r2Key, filename }.
  3. Accepted file types: PDF, PPTX only.                                                                  
  4. After upload, show the filename + a preview link (the R2 key resolves via existing signed URL         
  patterns).                                                                                               
  5. "Accept brief" button → POST /api/lb-slides/factory/runs/:id/accept-brief → status becomes            
  brief_ready.                                                                                             
  6. After acceptance, the tab shows "Brief accepted — waiting for Lorenzo to process" (Tab 2 is shell CC
  territory, not UI).                                                                                      
   
  ---                                                                                                      
  Tab 3 — Properties                                              
                                                                                                           
  Gate: show when status === "ingested" (Lorenzo has finished).   

  Form with 4 property pickers — one for each of slides 1, 2, 3, 5 (slide 4 is portfolio grid              
  auto-generated, slide 6 is income statement auto-generated — no picker needed).
                                                                                                           
  Use the existing list_properties / get_property API or whatever property list endpoint is already wired  
  in the LbSlides page.
                                                                                                           
  On save: POST /api/lb-slides/factory/runs/:id/properties with:                                           
  {
    "slide1PropertyId": 42,                                                                                
    "slide2PropertyId": 17,                                       
    "slide3PropertyId": 42,
    "slide5PropertyId": 88                                                                                 
  }                       
                                                                                                           
  All four fields are optional — send only the ones the admin has selected. The server validates ownership.
                                                                                                           
  After save, show "Properties saved — waiting for Lucca to draft" (Tab 4 is shell CC).

  ┌──────────────┬───────────────────────────────────────────────┐

  ├──────────────┼───────────────────────────────────────────────┤

  ├──────────────┼───────────────────────────────────────────────┤

  ├──────────────┼───────────────────────────────────────────────┤                                         
  │ ingested     │ Tab 3 (assign properties)                     │
  ├──────────────┼───────────────────────────────────────────────┤                                         
  │ drafting     │ Tab 4 indicator (Lucca running — read-only)   │
  ├──────────────┼───────────────────────────────────────────────┤                                         
  │ draft_review │ Tab 4 (approve slots)                         │
  ├──────────────┼───────────────────────────────────────────────┤                                         
  │ building     │ Tab 5 (agents running)                        │
  ├──────────────┼───────────────────────────────────────────────┤                                         
  │ complete     │ Tab 6 (download)                              │
  ├──────────────┼───────────────────────────────────────────────┤

  └──────────────┴───────────────────────────────────────────────┘
