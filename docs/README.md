The `.claude/` directory contains the AI development knowledge base:
- `.claude/skills/` — Domain knowledge organized by area (see [skill index](../.claude/skills/_index.md))
- `.claude/rules/` — Behavioral constraints for AI-assisted development
- `.claude/tools/` — Deterministic calculation tool schemas
- `.claude/manuals/` — User manual and checker manual content
